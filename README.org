* FlagRa

** Video Demo: [[https://www.youtube.com/watch?v=7ajQ-irD1VU]]

** Description

FlagRa is a flag guessing game, written in flask using the [[https://restcountries.com/][restcountries API]].


** Installation & Usage

#+begin_src shell
  source .venv/bin/activate
  pip install -r requirements.txt
  flask run
#+end_src

Then just open your browser at [[http://localhost:5000/]] and have fun!
